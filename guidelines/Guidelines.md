/// Fase 1

Crie um novo projeto com base em tailwind, shadcn, react que seja uma plataforma (hub de aplicações) para gestão de entidades médicas e eventos, no formato SAAS multitenant que acompanhe o associado (médicos) em sua jornada profissional.

Preciso que crie a estrutura do projeto, acesso ao banco e arquitetura básica para um projeto com essa caracteristica. 

Crie uma pasta com as diretrizes (arquivo MD) técnicas que devem ser seguidas no projeto quanto a estrutura de infra, segurança, arquitetura de software (DDD, Clean arch e clean code), testes automatizados que devem ser executados pela propria IA (criar tutorial para o humano poder executar).

A cada nova instrução deve ser organizado um arquivo MD com esta atualização de contextos de desenvolvimento do produto.

Use dados mokados para exemplificar a navegação. Ao realizar cadastro o novo cliente da plataforma deve salvar os dados no navegador para seguir com contrataçao de modulos, uso e etc.


³ Landing page #
Crie uma landingpage para o SAAS do software de gestao de eventos e entidades médicas da itarget tecnologia.

Crie seção para os produtos:
Gestão associativa
Gestão de eventos (módulo de inscrições, caex - gestão de expositores e estandes, programação científica do evento e palestrantes, submissão de trabalhos com multiplas fases, distribuição cega e automatica para os avaliadores)
EAD/LMS
Gesão de pontuação e cashback
Prova de título e exames
Módulo de formulários (quiz, pesquisas e avaliações, enquetes, formulários complementares de cadastro) 
Gestão de usuários (grupos e níveis de acesso com prazos de renovação, controle de acesso aos dados - LGPD - transição de diretoria seguro)

Crie o texto de engajamento e promocional para cada produto com foco em gestão associativa simplificado, melhoria das parcerias, melhoria do engajamento dos associados ofertando beneficios reais, melhor rentenção dos associados.


#UI e estilo#
Cores suaves, não muito berrantes;
Ambiente corporativo médico;
Personas entre 23 a 70 anos.
Propor uma UI clean, tipo google. foco no simples. White space.... sem mta informação na tela.
